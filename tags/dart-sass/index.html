Sass：@import rule

發表於 2020-05-17 | 留言
分類於 Web Dev

Sass 的 @import rule 可以引入 Sass 和 CSS stylesheet、提供對 mixin、function 和變數的存取，並且還能將多個 stylesheet 的 CSS 組合在一起。例如：main.scss 內使用 @import 引入 Sass 和 CSS 檔，在編譯 Sass 後，就只會產生一個 main.css 檔。
